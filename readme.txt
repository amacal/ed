*** ED Version 1.2 ***

Die Kompilierung von ED kann über eine der vier Stapeldateien erfolgen:

  bbc.cmd: Kompilierung über Borland C
  blc.cmd: Kompilierung über LCC
  bmc.cmd: Kompilierung über Microsoft Visual C
  bpc.cmd: Kompilierung über Pelles C

Achtung: Vor dem Lauf einer Stapeldatei müssen die Pfade zu den Bibliotheken und Header-Dateien der Entwicklungsumgebungen angepaßt werden. Alternativ können Make-Dateien verwendet werden oder die Quelltexte in eine integrierte Entwicklungsumgebung geladen und kompiliert werden.
